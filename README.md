# SpaceX Falcon 9 Landing Prediction 🚀

## Project Overview
SpaceX advertises Falcon 9 rocket launches at a much lower cost than other providers because the first stage of the rocket can be reused.

The goal of this project is to analyze SpaceX launch data and predict whether the Falcon 9 first stage will land successfully.

Predicting landing success helps estimate the cost of launches and provides insights into reusable rocket technology.

---

## Problem Statement
If the first stage lands successfully, the cost of the launch is significantly reduced. This project builds a machine learning model to predict the landing outcome.

---

## Data Sources

### 1. SpaceX API
Launch data was collected using the SpaceX REST API.

### 2. Web Scraping
Additional launch information was collected from Wikipedia using web scraping.

## Machine Learning Models

The following models were trained to predict landing success:

- Logistic Regression
- Support Vector Machine
- Decision Tree
- K-Nearest Neighbors

The models were evaluated using accuracy scores and confusion matrices.

---

## Results

The analysis shows that several factors influence landing success:

- Launch Site
- Payload Mass
- Orbit Type
- Year of Launch

Among the tested models, the Decision Tree model performed the best.
